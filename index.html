Quem é Rian

Meu nome é Rian Ferreira de Lima, tenho 16 anos e moro no Jardim Santa Rosa, em Campinas - SP. Atualmente curso Técnico em Informática, no 2º ano do Ensino Médio, e é nesse período que venho descobrindo o quanto gosto de tecnologia.

Comecei estudando lógica de programação, HTML, CSS e JavaScript na escola e logo percebi que queria ir além da sala de aula. Hoje estudo por conta própria para entender como sites e sistemas funcionam por trás das telas — este portfólio é um dos primeiros passos dessa jornada, construído enquanto aprendo a desenvolver do zero.

Meu objetivo é me especializar em desenvolvimento web e programação, construir projetos cada vez mais completos e, no futuro, atuar profissionalmente na área de TI — unindo lógica, design e código para criar soluções que façam sentido para quem usa.
